Epic Seven Trivia

Created By: Trever Edwards

Instructions

In this game you are given a question and 4 answers, if you pick the correct answer you will get 1 point. The game is out of 20 questions, and is completely randomized, so it will be in a different order everytime. If you run out of time to answer the question, it will be considered wrong. You can collect bookmarks if you get three questions in a row right. Each bookmark is worth 3 points. Good luck and have fun!
